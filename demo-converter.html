🎨 Convertisseur d'Image pour WPlace Bot

Transformez n'importe quelle image en pixel art pour l'utiliser automatiquement sur wplace.live !

📋 Comment Utiliser

  1. Ouvrez wplace.live dans votre navigateur
  2. Ouvrez la Console (F12 → onglet "Console")
  3. Chargez d'abord le bot (code ci-dessous)
  4. Utilisez les outils pour créer/convertir votre image
  5. Collez le script généré dans la console
  6. Exécutez wplaceBot.start() pour commencer à dessiner

⚠️ IMPORTANT - Charger le Bot d'abord

AVANT d'utiliser n'importe quel outil, vous DEVEZ charger le bot dans la console de wplace.live :

fetch('https://raw.githubusercontent.com/gcampos04/wplace-automation/main/wplace-bot.js').then(r=>r.text()).then(eval)

✅ Attendez de voir le message "🎨 WPlace Bot chargé !" et le panneau apparaître dans le coin de l'écran.

⚡ Démarrage Rapide

1. Ouvrez wplace.live et appuyez sur F12 (Console)

2. Collez ce code EN PREMIER pour charger le bot :

fetch('https://raw.githubusercontent.com/gcampos04/wplace-automation/main/wplace-bot.js').then(r=>r.text()).then(eval)

3. Attendez que le panneau apparaisse, puis choisissez une option :

  • 🖼️ Upload direct : Cliquez sur "📁 Charger une Image" dans le panneau
  • 🔧 Convertisseur : Cliquez sur "🔧 Convertisseur" pour des images existantes
  • 🎨 Éditeur : Cliquez sur "🎨 Éditeur" pour dessiner depuis zéro
  • ❤️ Test rapide : Tapez wplaceBot.loadHeartImage(); wplaceBot.start();

🎯 Conseils Importants

  • ⚠️ Chargez TOUJOURS le bot AVANT d'utiliser un outil
  • Console : F12 → onglet "Console" (Chrome/Firefox/Edge)
  • Position : Configurez X et Y où vous voulez commencer (ex : 100, 100)
  • Taille : Maximum 50x50 pixels pour un dessin rapide
  • Délai : Minimum 1000ms entre chaque pixel (1 seconde)
  • Test : Commencez avec des images petites
  • Arrêt : Tapez wplaceBot.stop() pour arrêter
